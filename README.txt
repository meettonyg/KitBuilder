A WordPress Media Kit Builder plugin with guest-first design using Nuclear Efficiency Architecture.

== Description ==

Build professional media kits with a modern drag-and-drop interface. Features include:
- Guest-first design (no registration required)
- React-powered interface
- Template system
- PDF export
- WP Fusion integration
- Pods custom fields

== Installation ==

1. Upload the plugin files to /wp-content/plugins/media-kit-builder/
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Media Kit Builder in your WordPress admin

== Requirements ==

- WordPress 5.0+
- PHP 7.4+
- Pods Framework (recommended)
- WP Fusion (recommended)
